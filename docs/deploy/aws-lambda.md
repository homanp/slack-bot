- Install Serverless:
    ```bash
    npm install -g serverless
    ```

- Make sure you have configured AWS profile. [See Docs](https://docs.aws.amazon.com/serverless-application-model/latest/developerguide/prerequisites.html)
  
- Run:
    ```bash
    npm install
    ```
    ```bash
    npm run build
    ```
    ```bash
    serverless deploy
    ```
